# Code of Conduct

## Our Pledge
We are committed to creating a welcoming and respectful community. By participating in this project, you agree to respect all contributors, maintainers, and users.

## Expected Behavior
- Be respectful and considerate in all interactions.  
- Be patient and understanding with beginners.  
- Use constructive feedback, not insults or personal attacks.  
- Report issues responsibly and politely.

## Unacceptable Behavior

## Scope
This Code of Conduct applies to all project spaces, including GitHub, issues, pull requests, discussions, and any project-related communication.
